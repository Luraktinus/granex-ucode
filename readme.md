## Installation

You can use the `cargo install` command:

    $ cargo install granex

or a classic build and run:

```bash
$ git clone https://github.com/dotcypress/granex
$ cd granex
$ cargo build --release
$ cp target/release/granex ~/.bin # assuming .bin is in your path
```
